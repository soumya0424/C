# C

C programming assignments and practice programs.

## Overview

This repository contains C programs for academic coursework and self-study. It includes solutions to programming exercises covering fundamental C concepts.

## Tech Stack

- C Programming Language
- GCC Compiler

## Topics Covered

- Variables, data types, and operators
- Control flow (if/else, loops, switch)
- Functions and recursion
- Arrays and pointers
- Structures and unions
- File handling
- Dynamic memory allocation

## Getting Started

```bash
# Compile a C file
gcc filename.c -o output

# Run the compiled program
./output
```
