## Overview

C is a collection of C programming assignments and practice programs covering fundamental concepts from variables to dynamic memory allocation.

**Why C?**

- 📐 **Core Concepts:** Variables, data types, control flow, functions, and recursion.
- 📂 **Structured Topics:** Organized into Computer Networks (cn) and Design & Analysis of Algorithms (daa).
- 🧠 **Memory Management:** Pointers, arrays, structures, and dynamic memory allocation.
- 📄 **File Handling:** Programs covering file I/O operations.
- ⚙️ **Compiled Programs:** All files compile and run with GCC.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** C
- **Compiler:** GCC

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/soumya0424/C
    cd C
    ```

### Usage

```bash
# Compile a C file
gcc filename.c -o output

# Run the compiled program
./output
```
